经典算法问题：编辑距离算法

编辑距离，又称Levenshtein距离（也叫做Edit Distance），是指两个字串之间，由一个转成另一个所需的最少编辑操作次数，如果它们的距离越大，说明它们越是不同。许可的编辑操作包括将一个字符替换成另一个字符，插入一个字符，删除一个字符。

例如将kitten一字转成sitting：

  1. sitten （k→s）
  2. sittin （e→i）
  3. sitting （→g）

俄罗斯科学家Vladimir Levenshtein在1965年提出这个概念。因此也叫Levenshtein Distance。

编辑距离递推公式

设字符串S，T长度分别为m, n，记S(i)为S从第1个字符到第i个字符之间的子串。S(0)表示空串，S(m)表示S本身。

因此，S和T间的编辑距离，可由S(i)和T(j)的编辑距离计算而来。

基础案例

  • 如果str1=”ivan”，str2=”ivan”，那么经过计算后等于 0。没有经过转换。相似度=1-0/Math.Max(str1.length,str2.length)=1
  • 如果str1=”ivan1”，str2=”ivan2”，那么经过计算后等于1。str1的”1”转换”2”，转换了一个字符，所以距离是1，相似度=1-1/Math.Max(str1.length,str2.length)=0.8

应用场景

  • DNA分析
  • 拼字检查
  • 语音辨识
  • 抄袭侦测

代码

Java 代码

private static int EditDistance(String[] sources, String[] targets) {
    int sourceLen = sources.length;
    int targetLen = targets.length;
    int[][] d = new int[sourceLen + 1][targetLen + 1];
    for (int i = 0; i <= sourceLen; i++) {
        d[i][0] = i;
    }
    for (int i = 0; i <= targetLen; i++) {
        d[0][i] = i;
    }

    for (int i = 1; i <= sourceLen; i++) {
        for (int j = 1; j <= targetLen; j++) {
            if (sources[i - 1] == targets[j - 1]) {
                d[i][j] = d[i - 1][j - 1];
            } else {
                //插入
                int insert = d[i][j - 1] + 1;
                //删除
                int delete = d[i - 1][j] + 1;
                //替换
                int replace = d[i - 1][j - 1] + 1;
                d[i][j] = Math.min(insert, delete) > Math.min(delete, replace) ? Math.min(delete, replace) :
                        Math.min(insert, delete);
            }
        }
    }
    return d[sourceLen][targetLen];
}

Python 代码

def edit_distance(sources, targets):
    len1 = len(sources);
    len2 = len(targets);
    dp = np.zeros((len1 + 1,len2 + 1))
    for i in range(len1 + 1):
        dp[i][0] = i;     
    for j in range(len2 + 1):
        dp[0][j] = j;

    for i in range(1, len1 + 1):
        for j in range(1, len2 + 1):
            delta = 0 if sources[i-1] == targets[j-1] else 1
            dp[i][j] = min(dp[i - 1][j - 1] + delta, min(dp[i-1][j] + 1, dp[i][j - 1] + 1))
    return dp[len1][len2]
